## The Beginning

CaviteNest started as a 4-member capstone project in 2024. The goal was simple: build a property rental and booking platform for Cavite. We used NestJS, TypeScript, and MongoDB as our tech stack.

## Going Solo

In 2026, I took over as the sole developer. This meant handling everything — backend architecture, database structure, deployment, and maintenance.

### Key Challenges

- **Scaling the database** — Restructured MongoDB schemas for better query performance
- **Authentication** — Implemented secure user auth with proper session management
- **Deployment** — Set up CI/CD with Vercel for automatic deployments

## Lessons Learned

1. Start with a solid architecture — it saves time later
2. Write documentation for yourself — you'll thank yourself in 3 months
3. Don't be afraid to refactor — better code leads to fewer bugs

## What's Next

I'm continuing to improve core platform features and expand beyond the original academic scope into a fully functional rental booking system.
