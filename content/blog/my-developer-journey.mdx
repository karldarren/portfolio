## Where It Started

My journey into tech started with hardware — diagnosing issues, reformatting systems, and setting up networks. As a freelance computer technician since 2023, I learned how to troubleshoot anything thrown at me.

## The Pivot to Web Development

During my BS Information Technology degree at Cavite State University, I discovered web development. Building my first Laravel application for Western Colleges was a turning point.

### The Tech Stack Evolution

- **2023** — HTML, CSS, basic JavaScript
- **2024** — PHP/Laravel, MySQL
- **2025** — React, Next.js, NestJS, MongoDB
- **2026** — TypeScript everywhere, deployment on Vercel & Hostinger

## What I Learned

The IT support background gives me an edge as a developer. I understand systems from hardware to software, from networking to deployment. It makes debugging easier and architecture decisions more informed.

## Advice for Others

If you're in IT support and want to transition to development — do it. Your troubleshooting skills transfer directly. Start with one language, build one project, and keep going.
